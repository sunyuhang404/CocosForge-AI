# 游戏策划案：CocosForge AI - 极客版

## 1. 游戏目标
- **游戏名称**: CocosForge AI
- **游戏类型**: 2D 平台跳跃游戏
- **目标玩家**: 喜欢挑战和解谜的玩家
- **游戏背景**: 玩家扮演一名勇敢的探险者，穿越各种险恶的关卡，最终找到宝藏。
- **胜利条件**: 完成所有关卡，找到宝藏。
- **失败条件**: 生命值耗尽或时间耗尽。

## 2. 核心玩法循环
- **关卡设计**: 每个关卡包含多个平台、障碍物和敌人。玩家需要通过跳跃、躲避和攻击来通过关卡。
- **资源收集**: 在关卡中收集金币和宝石，用于购买道具和升级角色能力。
- **敌人与战斗**: 遇到敌人时，玩家可以使用近战攻击或远程武器进行战斗。
- **谜题与机关**: 关卡中设置一些谜题和机关，玩家需要解开谜题才能继续前进。
- **生命值与能量**: 玩家有生命值和能量条，受到伤害会减少生命值，使用技能会消耗能量。

## 3. 关卡与难度
- **关卡设计**:
  - **关卡数量**: 10 个关卡
  - **关卡风格**: 不同的环境主题（如森林、沙漠、洞穴等）
  - **关卡难度**: 逐步增加，从简单到困难
  - **隐藏关卡**: 通过特定条件解锁隐藏关卡
- **难度调整**:
  - **敌人强度**: 随着关卡推进，敌人的攻击力和血量逐渐增加
  - **障碍物复杂度**: 后期关卡中的障碍物更加复杂，需要更高的操作技巧
  - **时间限制**: 某些关卡有时间限制，增加了紧迫感

## 4. 输入控制
- **键盘控制**:
  - **移动**: `A` 和 `D` 键控制左右移动
  - **跳跃**: `W` 键跳跃
  - **攻击**: `J` 键近战攻击，`K` 键远程攻击
  - **技能**: `Q` 键使用技能
- **鼠标操作**:
  - **无效**: 鼠标操作不生效，仅支持键盘控制
